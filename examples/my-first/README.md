# Example: my-first

A 10-second starter template — copy this folder, rewrite `script.json`,
and you have your first video.

## Run

```bash
./scripts/render.sh examples/my-first
```

Output: `out/full.mp4` (1080×1920).

## Compare

- `examples/my-first/` — minimal 3-slide template (this folder, copy and edit)
- `examples/schoger-demo/` — a real video that was actually shipped, kept as a worked example
